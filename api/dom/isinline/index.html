IsInline()

isInline(node) Since: 1.3.0

Checks if the passed node is inline or block level.

Syntax

var isInline = $.sceditor.dom.isInline(node[, includeCode]);

Parameters

node
Type: Node

The DOM node to check.

includeCode
Type: Boolean
Default: false

If to count the <code> tag as block level.

Return

Type: Boolean
