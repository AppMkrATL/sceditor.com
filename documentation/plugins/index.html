Plugins

Plugins can be enabled by adding them in the plugins option via the editors constructor.

See the custom plugins page for more information about creating plugins.

BBCode

Makes SCEditor output BBCode instead of HTML. For more information see the BBCode plugin page.

XHTML

Makes SCEditor output valid XHTML instead of HTML. For more information see the XHTML plugin page.
