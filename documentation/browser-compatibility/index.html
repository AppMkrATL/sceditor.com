Browser Compatibility

By default if a browser doesnâ€™t support WYSIWYG editing, the editor will be completely disabled leaving just the plain textarea. This only mobile browsers as all current desktop browsers support WYSIWYG editing.

You can set the editor to run in source mode if there is no WYSIWYG support by setting the runWithoutWysiwygSupport to true editors in the constructor.
