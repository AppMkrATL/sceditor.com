## saveRange()

{{% api_method name="saveRange()" since="1.3.5" %}}

Saveds the current range by inserting markers. The range be restored by the [restoreRange()](/api/rangehelper/restorerange/) method.


{{% api_section title="Syntax" %}}
```js
instance.saveRange();
```
